**The cost layer for AI-assisted development.**

Reads local Codex, Claude Code, Cursor, and Aider logs. Prints what each
PR cost. Offline. No login.

## Install in one line

```bash
uvx --isolated --from caliper-ai caliper
```

Python 3.11 or newer. The PyPI distribution is `caliper-ai`. The command and
the Python package are both `caliper`.

## What the first run prints

```bash
caliper overview
```

```text
Caliper - Overview
Vendors: claude-code (1,240 events) · openai-codex (860 events)

Last 7 days          $42
Last 30 days        $187
Last 90 days        $219

Events: 2,100
Cache savings: $640 at 72.4% cache hit
```

Names and numbers above are sanitized examples. No upload. No account.
The first run took eleven seconds on a cold cache.
