**The cost layer for AI-assisted development.**

Reads local Codex, Claude Code, Cursor, and Aider logs. Prints what each
PR cost. Offline. No login.

## Install in one line

```bash
uvx --from caliper-ai caliper
```

Python 3.11 or newer. The PyPI distribution is `caliper-ai`. The command and
the Python package are both `caliper`.

## What the first run prints

```bash
caliper overview
```

```text
Caliper - Overview
Vendors: claude-code (74,590 events) · openai-codex (20,500 events)

Last 7 days     48,727 credits     $3,383
Last 30 days    52,691 credits    $10,516
Last 90 days    52,691 credits    $10,897

Events: 95,090
Cache savings: $65,871 at 99.3% cache hit
```

Real numbers, one machine, one developer, ninety days. No upload. No
account. The first run took eleven seconds on a cold cache.
